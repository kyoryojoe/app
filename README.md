KyoRyoJoe/App
===========

はじめに
--------

KyoRyoJoeは地方自治体による直営橋梁点検の業務をサポートする[オープンソースソフトウェア](https://ja.wikipedia.org/wiki/オープンソースソフトウェア)です。
